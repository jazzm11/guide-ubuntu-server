Guide: Oppsett av Ubuntu Webserver (Apache / NGINX)

Steg-for-steg guide for ny lærling — opprett VM, installer Ubuntu Server, sett opp webserver og host denne guiden.

1. Opprette VM 2. Installere Ubuntu Server 3. Bruk SSH 4. Instalere NGINX 5. Lag og Hoste nettsiden

1. Opprett VM

• Gå til Proxmox: pve-backend.ikt-fag.no

• Trykk på Create VM øverst til høyre.

• Skriv navnet til VM-en og legg til din pool.

• Velg Ubuntu 24.04 som ISO image.

• Bytt Machine til q35 → trykk Next.

• Ikke endre noe på Disk.

• På CPU → endre kjerner til 4 og Type til host.

• Sett RAM til 4096 MiB.

• Hopp over Network.

• Trykk Finish.

2. Installer Ubuntu Server

• Start VM → velg Try or Install Ubuntu

• Velg språk: English

• Keyboard layout → Norwegian

• Velg Ubuntu Server for Installasjon

• Gå til Network Configuration → IPv4 → Method: Manual

Sett IP:

Subnet: 10.12.[pool].0/24
Address: 10.12.[pool].180
Gateway: 10.12.[pool].1
DNS: 10.12.[pool].255, 1.1.1.1
Search domain: ikt-fag.no

• Hopp over Proxy og Mirror.

• Velg Standard storage.

• Lag bruker og hostname.

• Skip Ubuntu Pro.

• Installer SSH.

• Reboot → trykk Enter.

3. Bruk SSH

Åpne Terminal på PC-en din og skriv:

ssh brukernavn@10.12.[pool].180

Svar yes og skriv passordet ditt.

4. Installer Webserver (NGINX)

Oppdater systemet og installer NGINX:

sudo apt update
sudo apt install nginx -y

Start NGINX:

sudo systemctl start nginx

Aktiver auto-boot:

sudo systemctl enable nginx

Sjekk status:

sudo systemctl status nginx

Test i nettleseren:

http://10.12.[pool].180

5. Lag og host nettsiden

Gå til webroot:

cd /var/www/

Gi deg selv eierskap:

sudo chown -R $USER:$USER /var/www

Klon git-repo:

git clone https://gitlab.ikt-fag.no/mcp/guide-ubuntu-server.git

Rediger NGINX config:

sudo nano /etc/nginx/sites-available/default

Endre:

root /var/www/html;

Til:

root /var/www/guide-ubuntu-server;

Restart NGINX:

sudo systemctl restart nginx

Test nettsiden:

http://10.12.[pool].180

Klone Repo fra Gitlab/GitHub

git clone [lenke til repo eks: https://gitlab.ikt-fag.no/mcp/guide-ubuntu-server.git]
